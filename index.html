Installation

With Bower

$ bower install ruban --save

Getting started

<!DOCTYPE html>
<html>
<head>
  <title>I &lt;3 Ruban</title>
  <meta charset="utf-8" />
  <link rel="stylesheet" href="components/normalize-css/normalize.css" />
  <link rel="stylesheet" href="components/font-awesome/build/assets/font-awesome/css/font-awesome.min.css" />
  <link rel="stylesheet" href="components/highlightjs/styles/tomorrow.css" />
  <link rel="stylesheet" href="components/ruban/dist/ruban.min.css" />
  <link rel="stylesheet" href="components/ruban/dist/ruban-print.min.css" media="print" />
</head>

<body>
  <section>
    <h1>My Awesome Presentation</h1>
  </section>

  <script src="components/jquery/jquery.min.js"></script>
  <script src="components/keymaster/keymaster.min.js"></script>
  <script src="components/hammerjs/dist/hammer.min.js"></script>
  <script src="components/highlightjs/highlight.pack.js"></script>
  <script src="components/ruban/dist/ruban.min.js"></script>
  <script>
    new Ruban();
  </script>
</body>
</html>

Features

Table of Contents

  • Auto-generated Table of Contents
  • Selects sections with a h1 as unique child
  • Exclude sections with the no-toc class
<section class="toc">
</section>
<section>
  <h1>First title</h1>
</section>
<section>
  <h1>Second title</h1>
</section>

Bookmarkable URLs

  • Just add an id to the section!
<section id="about-dinosaurs">
  <h1>About dinosaurs</h1>
</section>

Pagination

  • When will this boring presentation end?
new Ruban({
  pagination: true
});

Steps

  • Make your point
  • Step by step
    • <section>
  <h1>Steps</h1>
  <ul class="steps">
    <li>Make your point</li>
    <li>Step by step</li>
  </ul>
</section>

Details

  • Add details & speaker notes to your slides
    • Hidden on the Ruban
    • Displayed in the printed version
<details open="open">
  <summary>Notes</summary>
  <p>
    You can write down your comments here.
    They will be available in the printed version.
  </p>
</details>
